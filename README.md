Aplicación Android nativa, clon de la aplicacion oficial del Hospital Italiano, orientada a la gestión de pacientes, diseñada para centralizar información médica, facilitar el acceso a datos relevantes y mejorar la interacción entre usuario y sistema.

La app permite autenticación de usuarios, visualización de información y manejo estructurado de datos en la nube.

## 🚀 Características

* 👤 Registro e inicio de sesión de usuarios
* 📄 Gestión de información de pacientes
* ☁️ Persistencia en la nube
* 🔄 Actualización dinámica de datos
* 📱 Navegación basada en Fragments
* ⚡ Interfaz clara y funcional

## 🧠 Tecnologías

* **Kotlin**
* **Android SDK (XML + Fragments)**
* **Firebase Authentication**
* **Firebase Firestore / Realtime DB**
* **ViewBinding**
* **Material Design**

## 🏗️ Arquitectura

El proyecto utiliza una estructura basada en componentes clásicos de Android:

* `Activities` → Punto de entrada
* `Fragments` → Pantallas principales
* `Firebase` → Backend y almacenamiento
* `UI XML` → Layouts

Esto permite:

* organización modular
* separación básica de responsabilidades
* fácil mantenimiento
